# @shed/utils

:hammer: 一些辅助类，或还不足以放在独立包中的类

## 安装

```bash
npm install @shed/utils
```
或使用 `yarn`

```bash
yarn add @shed/utils
```

## 包含的类

- Gameloop
- EventDispatcher
- Keyboard

## 实例

```typescript
console.log('todo');
```
